## Quick reference — Profile tab (admin)

| Button | Purpose |
|--------|---------|
| **Opportunity management** | Manual opportunities — create, edit, delete |
| **Statistics & Analytics** | Team-wide survey/calculator progress by user |
| **Reports** | KPIs for all reps (beta — figures may be inaccurate) |
| **Appointment cycle time** | How long appointments take, by rep |
| **Admin Panel** | Users, CRM user linking, system settings, win/loss |
| **Tools** | Fill survey placeholder images by opportunity ID |
| **Workflow Override** | Deep workflow and survey fixes for one opportunity |
| **Logout** | Sign out |

Sales reps only see **Reports** (beta) and **Logout** on Profile.
