Creativ Energy hosts a [Model Context Protocol (MCP)](https://modelcontextprotocol.io) server so AI assistants can search and read this knowledge base.

**MCP endpoint:** `https://creativenergy.mintlify.app/mcp`

## Available tools

| Tool name | Purpose |
|-----------|---------|
| `search_creativ_energy` | Semantic search — titles, links, excerpts |
| `query_docs_filesystem_creativ_energy` | Read pages (`cat`, `head`), grep (`rg`), explore structure (`tree`, `ls`) |

### Search documentation (`search_creativ_energy`)

Semantic search across the knowledge base. Use for broad questions like "how do I complete the survey?" or "what outcomes can I record?".

Returns titles, links, and relevant excerpts from documentation pages.

### Query docs filesystem (`query_docs_filesystem_creativ_energy`)

Read-only access to documentation pages. Use when you need:

- Full text of a specific page (`head -80 /sales/appointment-workflow.mdx`)
- Exact keyword matches (`rg -il "energy bill" /sales/`)
- Site structure (`tree / -L 2`)

Each call is stateless — always use absolute paths starting with `/`.

## Cursor setup

Add to your project's `.cursor/mcp.json`:

```json
{
  "mcpServers": {
    "creativ-energy-knowledge-base": {
      "url": "https://creativenergy.mintlify.app/mcp"
    }
  }
}
```

Restart Cursor after saving. You can then ask the agent to search the knowledge base when helping with sales workflows or app questions.

## Example prompts

- "Search the Creativ Energy docs for survey photo requirements"
- "What are the solar workflow steps for sales reps?"
- "Read /sales/troubleshooting.mdx and help me fix a signing issue"

## Workflow for AI assistants

1. Start with **`search_creativ_energy`** for conceptual questions
2. Use **`query_docs_filesystem_creativ_energy`** to read full pages or grep for exact terms
3. Link users to published pages: `https://creativenergy.mintlify.app` + path (without `.mdx`)

Example: `/sales/getting-started.mdx` → [creativenergy.mintlify.app/sales/getting-started](https://creativenergy.mintlify.app/sales/getting-started)

## Syncing docs from this repo

The frontend repo keeps a `mintlify/` copy of documentation. To pull the latest published content from Mintlify:

```bash
npm run sync:mintlify
```

Publish changes to Mintlify first, then sync — or edit `mintlify/` locally and deploy through your Mintlify project.

## Security

The MCP server is read-only. It cannot modify documentation, access your machine, or call the Creativ Solar app API.
